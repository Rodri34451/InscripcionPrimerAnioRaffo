# InscripcionPrimerAnioRaffo

### Español
Proyecto final de 3er año de mi carrera en la Tecnicatura en Analista de Sistemas que consta de un sistema de inscripción para alumnos ingresantes de primer año a las carreras.
Este proyecto contiene únicamente los módulos asignados a mí (Logueo admin y Exportar datos a Excel). Contiene la base de datos junto con las consultas para realizar las operaciones

## Tecnologias utilizadas

- C# (Backend)
- HTML5
- JS puro
- Axios (para consultas a la API)
- Dapper (para convertir los resultados de las consultas en objetos)
- T-SQL
- ClosedXML (para convertir datos extraídos de tablas en hojas de cálculo)

### English
Final project of my 3rd year of the Technical Degree in System Analyst. It consists of a registration system for students entering their first-year in one of the careers.
This project only has the modules I was assigned to do (Logueo admin and Exportar datos a Excel). Comes with the database used during the project to realize all the operations.

## Technologies used

- C# (Backend)
- HTML5
- Vanilla JS
- Axios (for API requests)
- Dapper (to turn queries' results into objects)
- T-SQL
- ClosedXML (to turn data from tables into Worksheets)
